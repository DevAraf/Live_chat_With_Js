# Live_chat_With_JS

## Live Chat with JavaScript

A lightweight browser-based live chat interface built using **HTML**, **CSS**, and **JavaScript**. This project demonstrates interactive UI development and frontend programming skills with real-time chat simulation.

---

## Project Overview

This project simulates a basic live chat interface where users can type messages and see them display instantly in the chat area. It focuses on frontend interaction logic using pure JavaScript without backend dependencies. This is ideal as a beginner-to-intermediate example of DOM manipulation and event-driven programming.

---

## Key Features

- **Interactive Chat UI** – Messages are added live to the chat window  
- **Real-Time Simulation** – Instant message display without page reload  
- **User Input Handling** – Accepts and displays user text input  
- **Clean Frontend Logic** – Built with vanilla JavaScript

---

## Technologies Used

- **HTML** – UI structure  
- **CSS** – Basic styling  
- **JavaScript** – UI interaction and logic
